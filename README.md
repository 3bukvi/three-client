# ТРИ БУКВЫ VPN

Стильное и удобное Android-приложение для быстрого, стабильного и понятного VPN-подключения.

`ТРИ БУКВЫ VPN` создано для тех, кто хочет просто открыть приложение, выбрать сервер и подключиться без лишней настройки и перегруженных интерфейсов.

## Что это за приложение

`ТРИ БУКВЫ VPN` помогает:

- подключаться к VPN в один тап
- выбирать подходящий сервер вручную
- видеть состояние подключения и трафик
- управлять подпиской прямо в приложении
- получать поддержку без перехода в сторонние сервисы

Приложение сочетает простой интерфейс, быстрый доступ к подключению и полезные сетевые настройки для более гибкого использования.

## Возможности

- Быстрое подключение к VPN
- Выбор сервера из доступного списка
- Сохранение выбранного сервера
- Отображение статуса подключения
- Информация о входящем и исходящем трафике
- Просмотр статуса подписки и срока действия
- Активация пробного периода
- Покупка и продление подписки
- Вход через Telegram или Email
- Поддержка через встроенные тикеты
- Реферальная система
- Автоподключение
- Настройка DNS
- Split tunneling
- Настройка MTU
- Kill Switch

## Почему это удобно

- Понятный интерфейс без лишней перегрузки
- Быстрый доступ к подключению с главного экрана
- Удобное управление сервером и подпиской в одном месте
- Современный визуальный стиль
- Поддержка прямо внутри приложения

## Для кого подойдёт

Приложение подойдёт, если вам нужен VPN для:

- повседневного безопасного подключения
- стабильного доступа к нужным сервисам
- быстрого переключения между серверами
- простого управления подпиской без лишних действий

## Как установить

Если приложение распространяется через GitHub Releases:

1. Откройте страницу `Releases`
2. Скачайте последний `.apk`
3. Откройте файл на Android-устройстве
4. Разрешите установку из неизвестного источника, если Android это попросит
5. Установите приложение

После установки откройте приложение, войдите в аккаунт и выберите сервер для подключения.

## Что нужно для работы

- Android-устройство
- доступ к интернету
- аккаунт в сервисе
- активная подписка или пробный период

## Поддержка

Если возникли вопросы, проблемы с подключением или вопросы по подписке, в приложении есть встроенный раздел поддержки, где можно создать тикет и получить ответ.
